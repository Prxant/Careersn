The Problem & Our Solution

The Challenge

In today's competitive job market, finding a clean, fast, and user-friendly platform can be a challenge. Many existing portals are cluttered, slow, and offer a poor user experience.

The Solution

CareerSn is a full-stack web app built on the MERN stack to provide a seamless, modern, and localized experience for both job applicants and administrators in India.

Technology Stack

Frontend

  • ◆React & Vite
  • ◆Tailwind CSS
  • ◆Axios for API Calls

Backend

  • ◆Node.js & Express.js
  • ◆RESTful API

Database & Security

  • ◆MongoDB & Mongoose
  • ◆JWT (JSON Web Tokens)
  • ◆bcryptjs for Hashing
